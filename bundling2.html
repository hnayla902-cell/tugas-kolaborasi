Penyelesaian Sistem Persamaan Linear 3 Variabel

Harga Produk: Bansai, Lampu Nanas, dan Bingkai Kaca dari CD

1. Model Matematika

x = harga 1 Bansai

y = harga 1 Lampu Nanas

z = harga 1 Bingkai Kaca dari CD

Persamaan:

1) x + y + z = 170.000

2) 2x + y + z = 230.000

3) x + y + 2z = 215.000

2. Eliminasi Persamaan (1) dan (2)

(2x + y + z) − (x + y + z) = 230.000 − 170.000

→ x = 60.000

3. Substitusi x ke Persamaan (1)

x + y + z = 170.000 → 60.000 + y + z = 170.000

y + z = 110.000

4. Eliminasi Persamaan (3) dan Persamaan Baru

Pers 3: x + y + 2z = 215.000 → 60.000 + y + 2z = 215.000

y + 2z = 155.000

Kurangi persamaan y + 2z − (y + z) = 155.000 − 110.000

→ z = 45.000

5. Substitusi Nilai z ke Persamaan y + z = 110.000

y + 45.000 = 110.000

y = 65.000

✔ Hasil Akhir

(x, y, z) = (60.000, 65.000, 45.000)
