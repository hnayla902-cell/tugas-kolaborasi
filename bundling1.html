Penyelesaian Sistem Persamaan Linear 3 Variabel

Metode Eliminasi dan Substitusi

1. Persamaan Diketahui

1) 2x + y + z = 135.000

2) x + 3y + z = 185.000

3) 2x + 2y + 3z = 245.000

2. Eliminasi Persamaan (1) dan (2)

(2x + y + z) − (x + 3y + z) = 135.000 − 185.000

x − 2y = −50.000

→ Persamaan (4)

3. Eliminasi Persamaan (2) dan (3)

Kali 3 pada persamaan (2):

3x + 9y + 3z = 555.000

Kurangi dengan persamaan (3):

(3x + 9y + 3z) − (2x + 2y + 3z) = 555.000 − 245.000

x + 7y = 310.000

→ Persamaan (5)

4. Eliminasi Persamaan (4) dan (5)

Pers (4): x − 2y = −50.000 → ×7 → 7x − 14y = −350.000

Pers (5): x + 7y = 310.000 → ×2 → 2x + 14y = 620.000

Jumlahkan kedua persamaan:

9x = 270.000

x = 30.000

5. Substitusi Nilai x ke Persamaan (4)

30.000 − 2y = −50.000

−2y = −80.000

y = 40.000

6. Substitusi x dan y ke Persamaan (1)

2(30.000) + 40.000 + z = 135.000

100.000 + z = 135.000

z = 35.000

✔ Hasil Akhir

(x, y, z) = (30.000, 40.000, 35.000)
